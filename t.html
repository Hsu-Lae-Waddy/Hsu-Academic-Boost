2. Designer Screen

The Designer Screen is used to build the app interface.

Part Function
Palette Contains components
Viewer Shows app screen
Components Lists app components
Properties Changes component settings

3. Blocks Editor

Blocks are used to program the app behavior.

Event Blocks

Used when something happens like clicking a button.

Logic Blocks

Used for conditions like IF statements.

Math Blocks

Used for calculations and numbers.

🚀 MIT App Inventor Step-by-Step Tutorial

Step 1 — Open MIT App Inventor

  1. Open your web browser.
  2. Go to appinventor.mit.edu.
  3. Click Create Apps!
  4. Login using your Google account.

Step 2 — Create a New Project

  1. Click Projects.
  2. Click Start New Project.
  3. Type your project name.
  4. Press OK.
Example Project Name:
HelloApp

Step 3 — Add Components

Drag components from the Palette into the Viewer.

📝

Label

Displays text on the app screen.

🔘

Button

User clicks the button.

🖼️

Image

Shows pictures in the app.

Step 4 — Change Properties

Click a component and change its properties.

  • Button Text
  • Background Color
  • Font Size
  • Width and Height

Step 5 — Open Blocks Editor

  1. Click the Blocks button at the top right.
  2. The Blocks Editor will open.
  3. Use drag-and-drop blocks to create app behavior.

Step 6 — Create Button Click Event

Make the button change the label text. 

When Button1.Click
set Label1.Text to "Hello World"

What happens?

  1. User clicks Button1
  2. The event block runs
  3. Label text changes

Step 7 — Test Your App

  1. Install MIT AI2 Companion on your phone.
  2. Click Connect.
  3. Select AI Companion.
  4. Scan the QR code.
  5. Your app will run on your phone.

Step 8 — Build APK File

  1. Click Build.
  2. Select Android App (.apk).
  3. Wait for APK generation.
  4. Download the APK file.
  5. Install the app on Android.
