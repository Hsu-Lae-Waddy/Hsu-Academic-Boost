Loops

Loops repeat code execution. Two main types: for and while.

For Loop (Known Iterations)

# Count 0 to 4 for i in range(5): print(i) # 0, 1, 2, 3, 4 # range(start, stop, step) for i in range(1, 11, 2): print(i) # 1, 3, 5, 7, 9

While Loop (Condition-Based)

count = 0 while count < 5: print(count) count += 1 # 0, 1, 2, 3, 4

Loop Controls

for i in range(10): if i == 5: break # Exit loop if i == 3: continue # Skip iteration print(i)
