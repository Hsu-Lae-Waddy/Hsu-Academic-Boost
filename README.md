# 🚀 Hsu Academic Boost

**Hsu Academic Boost** is an educational platform designed to help students learn coding, computer science, and technology through simple, interactive, and beginner-friendly lessons.

Our mission is to make technology education accessible, engaging, and practical for students of all ages.

---

## 📚 Featured Lessons

### 🧩 Scratch Programming

Learn computational thinking, logic, and game development through block-based programming.

### 🐍 Python Coding

Learn Python from beginner to intermediate level:

* Variables
* Data Types
* Strings
* Conditions
* Loops
* Functions
* Lists, Tuples, Sets, Dictionaries
* File Handling
* Projects

### 🐍 Python OOP Concepts

Learn Object-Oriented Programming:

* Classes & Objects
* Encapsulation
* Inheritance
* Polymorphism
* Abstraction

### 🌐 Web Basics

Introduction to:

* HTML
* CSS
* JavaScript
* Responsive Design

### 👩🏻‍💻 Computer Science

Learn core computer science concepts:

* Computer Fundamentals
* Operating Systems
* Algorithms
* Data Structures
* Software Development

### 📶 Networking

Learn networking fundamentals:

* Internet Basics
* IP Addresses
* Routers & Modems
* LAN, MAN, WAN
* WiFi & Mobile Data

### 📱 MIT App Inventor

Build mobile applications using visual programming.

### ▶️ PowerPoint

Learn presentation design and communication skills.
